# Jira Sprint Guidelines

## Forecast Rationale

Base level story point for simple tasks will be 2 story points. We determine a simple task by how straightforward it or by how quick it is to finish. For example, initializing the project to create the scaffolding (which is usially an npm insall command) would be 2 story points.
Another would be creating a simple top navigation bar. While this is not as simple, it is still a straightforward task.
